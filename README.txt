How to use:
1. Open the folder "KneatCodeChallenge\bin\Debug" and run the app "KneatCodeChallenge.exe".
2. When the program start please inform the distance between 2 planets in MGLT and press "ENTER".
3. The program will show you how many stops each starship needs to take at the distance informed.
4. If you should to make another simulation press Y or press another key to exit the program.

How to view the code:
1. Please open the file "KneatCodeChallenge.sln" with the Visual Studio.
2. You can switch between the files in the "Solution Explorer".
